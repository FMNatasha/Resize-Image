# Editor Immagini Online

Editor professionale per ritagliare, ruotare e modificare immagini direttamente dal browser. Nessuna installazione richiesta, tutto viene elaborato localmente nel tuo browser per la massima privacy.

## 🎯 Caratteristiche

- **Caricamento facile**: Carica immagini tramite selezione file o drag & drop
- **Ritaglio personalizzato**: Area di ritaglio trascinabile e ridimensionabile con 8 maniglie
- **Dimensioni preset**: 
  - Fototessera (314×251 px)
  - Firma (50×251 px)
  - Dimensioni personalizzate
- **Rotazione**: 
  - Rotazione rapida a 90° e -90°
  - Rotazione precisa con ruota interattiva per regolazioni di qualsiasi grado
- **Specchiatura**: Flip orizzontale e verticale
- **Zoom**: Regolazione da 10% a 300%
- **Filtro bianco e nero**: Conversione istantanea in scala di grigi
- **Controllo dimensione file**: Imposta la dimensione massima in KB (default 200 KB)
- **Esportazione**: Salva in formato JPG o PNG

## 🚀 Utilizzo

1. Apri l'editor nel browser
2. Carica un'immagine (clicca "Scegli File" o trascina l'immagine nell'area centrale)
3. Applica le modifiche desiderate:
   - Seleziona o personalizza le dimensioni di ritaglio
   - Ruota l'immagine
   - Specchia orizzontalmente o verticalmente
   - Regola lo zoom
   - Applica il filtro bianco e nero se necessario
4. Esporta l'immagine nel formato preferito

## 🔒 Privacy

Tutte le operazioni vengono eseguite localmente nel tuo browser. Le immagini non vengono mai caricate su server esterni. I tuoi dati rimangono sul tuo dispositivo.

## 💻 Tecnologie

- HTML5 Canvas per l'elaborazione delle immagini
- JavaScript vanilla (nessuna dipendenza esterna)
- Design responsive compatibile con desktop e mobile
- Interfaccia drag & drop moderna
