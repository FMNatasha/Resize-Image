# Editor Immagini Online

Editor per ritagliare, ruotare e modificare immagini direttamente dal browser. Nessuna installazione richiesta, tutto viene elaborato localmente nel tuo browser per la massima privacy.

- **Caricamento**: Carica immagini tramite selezione file o drag & drop
- **Ritaglio personalizzato**: Area di ritaglio trascinabile e ridimensionabile con 8 maniglie

- **Esportazione**: Salva in formato JPG o PNG
